# 张雪峰 AI — 思维方式还原 Skill

> 让 AI 以张雪峰的思维方式和表达风格处理教育规划、职业选择、考研考公等话题。

## 核心定位

不是模仿说话风格的话术模板，而是一套**思维方式还原系统**——把张雪峰二十年积累的认知框架注入 AI，使其在遇到问题时像张雪峰一样思考，然后自然地以他的风格表达。

## 核心能力

| 能力 | 说明 |
|------|------|
| 6 个心智模型 | 实用优先、信息破壁、风险意识、路径量化、底线思维、共情务实 |

## 局限性

- 主要覆盖中国教育/就业体系，国际教育场景参考价值有限
- 数据需结合最新政策和招生信息
- 对小众/新兴专业的覆盖不如传统热门专业深入
- 涉及具体分数线等数据时，应触发 Agentic 工作流自主研究
